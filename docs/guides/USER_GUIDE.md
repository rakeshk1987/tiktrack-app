# TikTrack User Guide

Last updated: May 8, 2026

## Who This Is For

- Parents (role: `parent_admin`)
- Children (role: `child_user`)

## Quick Start

1. Sign up or log in.
2. Parent creates or links child account(s).
3. Parent sets routine/tasks/reminders/rewards.
4. Child completes quests and submits proof when required.
5. Parent reviews proofs, tracks progress, and approves redemptions.

## Parent Dashboard Menus

### 1. Dashboard

Use this as your daily control center:
- overall progress
- quick status
- recent activity
- child snapshots

### 2. Family

Manage family and child profiles:
- add child accounts
- link parent-child accounts
- maintain child profile basics (name, age-related info)

### 3. Tasks

Create and manage quests:
- add tasks with priority, difficulty, stars
- assign to child
- set due dates
- choose proof-required tasks

### 4. Automation

Configure smart systems:
- routine configuration
- task scheduler
- reminders setup
- default reminder seeding

### 5. Proofs

Review child submissions:
- approve/reject proof logs
- approved proofs contribute to progress/stars

### 6. Events

Track important events:
- add school/life events
- attach reminder lead days

### 7. Growth

Maintain growth entries:
- height and weight logs
- trend monitoring in analytics

### 8. Rewards

Configure motivation system:
- star conversion settings
- reward item catalog
- manage availability
- review redemption history

### 9. Exams

Record academic performance:
- subject-wise marks
- exam date history
- feeds weak-subject and planning logic

### 10. Challenges

Run parent-vs-child mini competitions:
- create challenge
- set target score
- track winner and completion

### 11. Communication

Use inbox messaging with child:
- send updates/encouragement
- review message thread

### 12. Settings

System and account controls:
- sign out
- account-level controls and utility actions

## Child App Menus

### 1. Home

Daily motivation + overview:
- recommendations
- mood check-in
- today highlights
- progress indicators

### 2. Quests

Task execution area:
- see assigned quests
- mark complete tasks
- upload proof where required

### 3. Diary

Private reflection space:
- write daily notes
- keep short personal entries

### 4. Profile

Progress view:
- stars
- streaks and shields
- personal stats view

## Suggested Daily Workflow

### Parent

1. Morning: review Dashboard + Automation reminders.
2. Afternoon: check Proofs and approve pending items.
3. Evening: review Rewards/Exams/Growth and plan next day.

### Child

1. Pick mood on Home.
2. Complete Quests.
3. Submit proofs if asked.
4. Write a short Diary entry.

## Tips

- Start with fewer tasks and increase gradually.
- Use proof-required tasks only when you need verification.
- Keep rewards meaningful but achievable.
- Use challenges occasionally to keep engagement high.

## Troubleshooting

- If data looks stale, refresh and check connection status indicators.
- If reminders do not appear, verify reminder is enabled and schedule time is correct.
- If proof upload fails, retry with a smaller/clear image and stable connection.
